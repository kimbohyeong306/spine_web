# 🎭 보형킴의 원화 아뜨리움 (Drawing Atrium)

Spine 2D를 활용한 캐릭터 애니메이션 및 리깅 작업물을 한눈에 확인할 수 있는 웹 포트폴리오 공간입니다.

> **🔗 라이브 데모:** [https://kimbohyeong306.github.io/spine_web/](https://kimbohyeong306.github.io/spine_web/)

---

## 🌟 Project Overview

이 프로젝트는 **Spine 2D**로 제작된 다양한 애니메이션 데이터를 웹 환경에서 인터랙티브하게 보여주기 위해 구축되었습니다. 단순한 영상 출력을 넘어, 웹 기술을 통해 캐릭터의 움직임을 실시간으로 감상할 수 있도록 구성했습니다.

## 🛠 Tech Stack

- **Animation:** Spine 2D (Esoteric Software)
- **Web:** HTML5, CSS3, JavaScript
- **Deployment:** GitHub Pages
- **Library:** Spine Web Player

## 🎨 Key Features

- **실시간 애니메이션 재생:** Spine Web Player를 사용하여 고해상도 텍스처와 부드러운 애니메이션을 웹에서 직접 구현.
- **다양한 모션 전시:** Idle, Attack, Walk 등 캐릭터의 상태별 애니메이션을 리스트 형태로 제공.
- **반응형 디자인:** 모바일 및 데스크톱 환경 모두에서 최적화된 뷰 제공.

## 📂 Project Structure

- `/assets`: 스파인 데이터 파일 (.skel, .json, .atlas, .png) 및 이미지 리소스.
- `/css`: 웹 페이지 레이아웃 및 스타일 시트.
- `/js`: Spine Web Player 연동 및 페이지 로직 스크립트.
- `index.html`: 메인 랜딩 페이지.
